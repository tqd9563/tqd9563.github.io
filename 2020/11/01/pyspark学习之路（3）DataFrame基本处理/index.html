        本节主要整理一下Spark DataFrame的基本数据处理方法。

打印/获取DataFrame的数据：show()/collect()

        Spark DataFrame与pandas df不同，单单输入一个df并回车是无法取得DataFrame中的数据的，要想打印/获取到真实的数据，有以下两种方法：

  • show()方法：将DataFrame的前n行打印在控制台上，默认n = 20
  • collect()方法：将DataFrame的所有记录合并成一个list返回， list的长度等于DataFrame的行数，每个分类对应DataFrame的一行，是一个pyspark.sql.Row类实例

        以上面的代码为例，Row(Id=123, Name='Katie', Age=19, Color='brown')这个Row对象就对应了DataFrame的第一行，Spark DataFrame的每一行都是一个Row对象。想要获取Row的字段信息有下面两种方式：

  • 属性的方式：row.key
  • 字典键的方式：row[key]

获取所有字段类型和字段名：dtypes/columns

  • dtypes属性：返回一个由(colName, dtype)组成的list
  • columns属性：返回所有列名组成的list

获取行数：count()

        count()方法可以返回DataFrame的行数

根据列名获取一列/多列：select(*cols)

        pandas所支持的通过属性和索引来获取列的方法，在spark中也同样适用：

  • 属性的方式：df.colName
  • 索引的方式：df[colName]

        但是需要特别注意的是：通过上面两种方式选出的一列是一个pyspark.sql.Column类实例对象，而Column对象是not callable的，即它无法被collect()，因此不像pandas在获取这个列后直接就能看到这个字段对应的数据，在spark是里是看不到的。当然如果用索引的方式选取多个列，其返回的结果就是一个DataFrame了，也就可以show()或是collect()。

根据列条件来过滤行：where(condition)/filter(condition)

        两种方法是等价的，where()是filter()的另一种别名写法。参数condition需要是一个字符串形式的SQL表达式，或者是一个布尔类型的Column对象。

删除一列/多列：drop(*cols)

        drop()方法会返回一个删除了传入列的新的DataFrame，但是原对象的列还是完整保留的。参数*cols可以传入多个列名字符串。

新增一列：withColumn(colName, col)

        withColumn()方法返回一个新加了一列的新的DataFrame。（这个方法一次只能新加一列）

  • colName：新的列名，string类型
  • col： 新列的一个Column表达式

重命名列

        Spark中对某一列重命名通常有下面三种方法：

  • withColumnRenamed(existing, new)：返回重命名后的新DataFrame，参数existing是旧名，new是改后的新名
  • selectExpr(*expr)：直接用sql语句的as语法来对列重命名
  • alias()：这是Column对象的方法， 需要在df的select()方法内部才能使用

        这三种方法中，后两种的限制比较大，如果只是写明重命名的列，最后返回的结果也只有这一列。如果想要把所有列都返回的话，需要显式的把所有列都写进selectExpr()或select()中。

按一列/多列排序：sort(*cols, **kwargs)/orderBy(*cols, **kwargs)

        两种方法是等价的，都可以对一列/多列进行排序。控制升序/降序排序的方法有两种：

  • ascending = True / False：显示传入参数控制前面的列是升序还是降序排序
  • col.asc() / desc()：传入列名的时候，用Column对象传，然后调用其asc()或desc()方法（这种写法更推荐）

去重：dropDuplicates(subset=None)/distinct()

        如果想要将所有完全相同的行给去掉，就用distinct()方法；而如果单独仅针对部分列，去除这部分列上的取值完全相同的行，则使用dropDuplicates(subset=None)方法，参数subset指定了要去重的列，需要特别注意，即使只指定一个列，也需要传入一个列名组成的list！
